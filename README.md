# Decimal and Binary Converter

A simple, interactive C++ command-line application that converts numbers between Decimal (base-10) and Binary (base-2) formats. 

---

## 🚀 Features

* **Decimal to Binary Conversion:** Enter a standard decimal number and generate its binary equivalent using division and remainders.
* **Binary to Decimal Conversion:** Enter a binary number (composed of 0s and 1s) and calculate its exact decimal value using place values.
* **Interactive Menu:** A user-friendly prompt that lets you choose exactly which conversion tool you need upon launching.

---

## 🛠️ How to Use

### Option 1: Running the Executable (.exe)
If you just want to use the program without messing with code, you can run the pre-compiled executable:
1. Download the provided `.exe` file from this repository.
2. Double-click the file to run it in your terminal or command prompt.
3. Follow the on-screen instructions:
   * Type **`1`** to convert Decimal to Binary.
   * Type **`2`** to convert Binary to Decimal.

### Option 2: Running from Source Code
If you want to view, modify, or compile the C++ code yourself:
1. Ensure you have a C++ compiler installed (such as GCC/MinGW).
2. Open your terminal and navigate to the folder containing the `.cpp` file.
3. Compile the code using the following command:
   ```bash
   g++ main.cpp -o converter
